# File Explorer Application
- A mini project developed in C++ to simulate a basic file explorer with essential file operations and search functionalities.
- The project is divided into multiple development phases (Day-1 to Day-5), each adding new features and improvements.

## Features
- Create, view, and navigate folders
- Search and manage files
- Perform basic file operations (create, read, delete)
- Modular structure for easy extension and maintenance
  
### Project Structure
- Day-1/ → Basic file explorer commands
- Day-2/ → Simple file explorer functionality
- Day-3/ → Interactive explorer in C++
- Day-4/ → File search implementation
- Day-5/ → Explorer info and demo modules

#### Tech Stack
Language: C++
Tools Used: Visual Studio Code, GitHub
Concepts: File handling, data structures, modular programming
